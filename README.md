CppUTest Cookbook
=================

Installs CppUTest from source

Scope
-----
This cookbook is concerned with the installation of CppUTest from source found at GitHub
(https://github.com/cpputest/cpputest)

Requirements
------------
- Git
- Build Essential
- Libtool
